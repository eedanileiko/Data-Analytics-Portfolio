# Data Analysis & Machine Learning Projects

Этот репозиторий содержит мои учебные проекты по **анализу данных** и **машинному обучению**, выполненные в рамках курсов по статистике, аналитике и ML.
Цель репозитория — продемонстрировать мой подход к работе с данными, умение формулировать гипотезы, выбирать модели и интерпретировать результаты.

Проекты оформлены с комментариями, визуализациями и выводами.

---

## Содержание репозитория

###  `Analytics_pandas.ipynb`

* Работа с данными с использованием **pandas**
* Очистка данных, преобразования, агрегации
* Первичный исследовательский анализ (EDA)
* Акцент на читаемость и воспроизводимость кода

---

###  `Hypothesis_testing.ipynb`

* Проверка статистических гипотез
* Использование t-test, non-parametric тестов
* Формулировка нулевой и альтернативной гипотез
* Интерпретация результатов и выводы
---

### `Linear_regressions_EDA_gradient.ipynb`

* Линейная регрессия: теория и практика
* Реализация градиентного спуска
* Анализ качества модели
* Визуализация результатов и ошибок

---

### [`Trees`](Trees_Danileyko.ipynb)

* Деревья решений
* Обучение и интерпретация модели
* Анализ переобучения
* Сравнение различных параметров модели

---

### `Features_prediction_ml_project_Danileiko`

* Небольшой ML-проект основанные на выгрузке данных из известной игры Dota
* Работа с признаками и поиск лучшей модели
* Обучение и оценка моделей
* Анализ важности признаков
